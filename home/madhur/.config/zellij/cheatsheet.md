# Zellij Cheatsheet

## Tabs

| Key | Action |
|-----|--------|
| `Alt 1-9` | Go to tab |
| `Ctrl t` → `n` | New tab |
| `Ctrl t` → `x` | Close tab |
| `Ctrl t` → `r` | Rename tab |
| `Alt i` / `Alt o` | Move tab left / right |
| `Ctrl t` → `h/j/k/l` | Navigate tabs |

## Panes

| Key | Action |
|-----|--------|
| `Ctrl p` → `r` | Split right |
| `Ctrl p` → `d` | Split down |
| `Ctrl p` → `s` | New stacked pane |
| `Ctrl p` → `x` | Close pane |
| `Ctrl p` → `f` | Fullscreen toggle |
| `Ctrl p` → `w` | Toggle floating panes |
| `Ctrl p` → `e` | Embed / float pane |
| `Ctrl p` → `i` | Pin pane |
| `Ctrl p` → `z` | Toggle pane frames |
| `Alt n` | New pane |
| `Alt f` | Toggle floating panes |

## Navigation

| Key | Action |
|-----|--------|
| `Alt h/j/k/l` | Move focus between panes |
| `Alt ←/→` | Move focus or switch tab |
| `Alt ↑/↓` | Move focus up / down |

## Resize (`Ctrl n`)

| Key | Action |
|-----|--------|
| `h/j/k/l` | Expand pane in direction |
| `H/J/K/L` | Shrink pane in direction |
| `Alt +` / `Alt -` | Resize from anywhere |

## Move Panes (`Ctrl h`)

| Key | Action |
|-----|--------|
| `h/j/k/l` | Move pane in direction |
| `n` / `p` | Move pane forward / back |

## Scroll (`Alt s`)

| Key | Action |
|-----|--------|
| `j` / `k` | Scroll down / up |
| `d` / `u` | Half page down / up |
| `Ctrl f` / `Ctrl b` | Full page down / up |
| `Ctrl c` | Scroll to bottom + exit |
| `s` | Enter search |
| `e` | Open scrollback in editor |

## Search (inside scroll)

| Key | Action |
|-----|--------|
| `n` / `p` | Next / previous match |
| `c` | Toggle case sensitivity |
| `w` | Toggle whole word |
| `o` | Toggle wrap |

## Session (`Ctrl o`)

| Key | Action |
|-----|--------|
| `w` | Session manager |
| `d` | Detach |
| `a` | About |
| `c` | Configuration |
| `p` | Plugin manager |
| `s` | Share session |

## Global

| Key | Action |
|-----|--------|
| `Ctrl g` | Lock / unlock mode |
| `Alt p` | Toggle pane in group |
| `Alt Shift p` | Toggle group marking |
| `Esc` / `Enter` | Exit current mode |

> **Note:** `Ctrl q` is unbound (conflicts with micro editor)
